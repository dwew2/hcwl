Browser requirements

These pages have been tested using Internet Explorer 5.5 or 6 for PC, and Internet Explorer 5.1 and Netscape 7 for Mac, but they should be viewable using any recent version of the common web browsers. Your browser needs to support frames.

Character encoding

You may need to change the encoding of the Character Set setting of your web browser to Unicode / Universal Alphabet (UTF-8) to view special characters in the texts properly. On Internet Explorer this is done by selecting the Encoding or Character Set option from the View menu.

If you still have problems viewing the texts, please let us know.
